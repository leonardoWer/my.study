# my.study
`Сайт для подготовки к тестам в формате карточек`

---

Проект поднимается локально и нужен для подготовки к тестам в формате карточек.
Все что нужно сделать - сохранить структуру карточек в формате жсон, а после, вставить в нужно поле.
Жсон парситься и генерируются карточки, и можно отвечать на случайный вопрос.
